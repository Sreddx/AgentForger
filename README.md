# AgentForger

Public minimal repository for the **AgentForger** OpenClaw agent package.

## Included
- `agents/agentforger/`
- `skills/agentforger/`

## Purpose
AgentForger is a meta-agent for OpenClaw focused on:
- creating agents and skills
- updating existing artifacts
- refactoring existing artifacts
- auditing agent/skill quality and structure

## Notes
This repository intentionally includes only the agent and its specific skill package, without the broader workspace or legacy Skillforge material.
